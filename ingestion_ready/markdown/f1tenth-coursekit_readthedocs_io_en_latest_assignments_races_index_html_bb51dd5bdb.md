# Races¶

There are 3 races:

  1. **Race 1** tests the students’ application of _reactive methods_.

  2. **Race 2** tests the students’ application of _pure pursuit_ and _map-based_ methods.

  3. **Race 3** culminates to head to head racing with another team’s car.

The first two races are timed trials. There are time brackets that correspond to grade brackets.
