# Race 2¶

**Track:** You want to choose a track that is a closed loop. At Penn, we have the students race around the hallways in the Levine building. It can be the same track Race 1 or a different track.

[![../../_images/levine_track.jpg](../../_images/levine_track.jpg)](../../_images/levine_track.jpg)

The hallways on the second floor of the Levine buildling form a rectangular track.¶

**Race Prep:** Teams should be given the track beforehand so they can map the track.

**Race Day:** The format is very similar to [Race 1](race1.html#doc-race1). View the [Race2 Rubric](https://docs.google.com/spreadsheets/d/1gvkIyaZuPUC6-Cb7FpFiT8ZrpwOfUKHX/edit?usp=sharing&ouid=116660630717436430566&rtpof=true&sd=true) and [Grading](../grading.html#doc-grading) to see how teams are evaluated. If time allows, there can be a practice session beforehand. If not, then have teams run their vehicle around the track and record each lap time and number of consecutive laps. There are two heats so teams will have two chances to complete the race.
