# Start Here!¶

There is enough material for a full semester course, however if you have a limited amount of time, here are some suggestions:

  1. **4 Weeks** \- Go through [Module A](../lectures/ModuleA/index.html#doc-modulea) and [Module B](../lectures/ModuleB/index.html#doc-moduleb) and complete [Lab 1](../assignments/labs/lab1.html#doc-lab1), [Lab 2](../assignments/labs/lab2.html#doc-lab2), [Lab 3](../assignments/labs/lab3.html#doc-lab3), and [Lab 4](../assignments/labs/lab4.html#doc-lab4). This will be enough to get your vehicle race ready.

  2. **10 Weeks (~1 Quarter)** \- Go through [Module A](../lectures/ModuleA/index.html#doc-modulea), [Module B](../lectures/ModuleB/index.html#doc-moduleb), [Module C](../lectures/ModuleC/index.html#doc-modulec), and [Module D](../lectures/ModuleD/index.html#doc-moduled) and complete [Lab 1](../assignments/labs/lab1.html#doc-lab1), [Lab 2](../assignments/labs/lab2.html#doc-lab2), [Lab 3](../assignments/labs/lab3.html#doc-lab3), [Lab 4](../assignments/labs/lab4.html#doc-lab4), [Lab 5](../assignments/labs/lab5.html#doc-lab5), and [Lab 6](../assignments/labs/lab6.html#doc-lab6). This will teach you how to race efficiently.

  3. **15 Weeks (~ 1 Semester)** \- Go through all of the [Lectures](../lectures/index.html#doc-lecture-intro) and complete all of the [Labs](../assignments/labs/index.html#doc-labs-intro). This is the most comprehensive learning experience.

This documentation page does not contain any build or software instructions. See the [Build Page](http://f1tenth.org/build.html) for that information.

If you are looking for the old build/code page, you can find it [here](https://f1tenth.github.io/build-old.html). Please note that we no longer support any of this material and the majority of the content is outdated.
