# Lecture 3 - Rigid Body Transformation

## Overview:

A point in the real world can be defined in different ways from different perspectives. If a lidar, attached to a moving vehicle, detects an obstacle, we might want to know the obstacle’s position in the real world. As the vehicle traverses the environment to create a map, the algorithm on the vehicle creates multiple submaps and then ties them together to form one global solution. Both applications require understanding and using reference frames and frame transformations. This lecture covers these topics.

## Topics Covered:

*   Coordinate Frames and Reference Frames
*   Rigid Body Transformations
*   Frames and Transformations in ROS
