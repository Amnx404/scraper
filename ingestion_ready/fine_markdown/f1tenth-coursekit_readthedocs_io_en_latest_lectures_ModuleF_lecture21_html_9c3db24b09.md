# Lecture 21 - Model Predictive Control

**Overview:**

This lecture covers Model Predictive Control (MPC). MPC is a control strategy that uses a model to predict future states. The linear version of MPC uses a linearized bicycle model to represent the RoboRacer vehicle. The nonlinear version is more complex, incorporating additional variables and parameters.

**Topics Covered:**

*   Linear MPC
*   Non-linear MPC
