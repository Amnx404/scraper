# ABOUT

AutoDRIVE is an integrated platform for autonomous driving research and education, bridging simulation and hardware deployment with the AutoDRIVE Simulator and AutoDRIVE Testbed for sim2real applications. It also includes the AutoDRIVE Devkit for rapid algorithm development and supports smart-city solutions for traffic management.

## AutoDRIVE Testbed

The AutoDRIVE Testbed is a hardware setup featuring a scaled vehicle model named Nigel and a modular infrastructure development kit. Nigel is equipped with redundant sensors, actuators for motion control, and a lighting system. It can be teleoperated or self-driven. The infrastructure kit includes environment modules and traffic elements.

## AutoDRIVE Simulator

The AutoDRIVE Simulator is the digital twin of the AutoDRIVE Testbed, built on the Unity game engine. It allows virtual prototyping of algorithms and communicates with autonomy algorithms via a WebSocket interface. The standalone application supports cross-platform deployment (Windows, macOS, Linux) at Full HD resolution (1920x1080p) and is designed for efficient resource utilization, supporting both single-machine and distributed computing setups.

*   **Source Branch:** [AutoDRIVE Simulator](https://github.com/Tinker-Twins/AutoDRIVE/tree/AutoDRIVE-Simulator)
*   **Latest Release:** [AutoDRIVE Simulator 0.3.0](https://github.com/Tinker-Twins/AutoDRIVE/releases/tag/Simulator-0.3.0)
*   **Upcoming Release:** AutoDRIVE Simulator 0.4.0

## AutoDRIVE Devkit

The AutoDRIVE Devkit enables users to leverage the AutoDRIVE Simulator and Testbed for rapid development of autonomy algorithms. It includes the ADSS Toolkit for autonomous driving and the SCSS Toolkit for smart city management. The Devkit supports local and distributed computing, is compatible with ROS, and offers direct scripting support for Python and C++.

*   **Source Branch:** [AutoDRIVE Devkit](https://github.com/Tinker-Twins/AutoDRIVE/tree/AutoDRIVE-Devkit)
*   **Latest Release:** [AutoDRIVE Devkit 0.3.0](https://github.com/Tinker-Twins/AutoDRIVE/releases/tag/Devkit-0.3.0)
*   **Upcoming Release:** AutoDRIVE Devkit 0.4.0
