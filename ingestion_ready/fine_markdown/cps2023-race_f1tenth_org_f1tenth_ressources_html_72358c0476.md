# F1TENTH Resources

## Build

The F1TENTH Autonomous Vehicle System is an open-source platform for autonomous systems research and education on a 1:10 scale. It serves as the baseline for the in-person competition, providing sensors and computation power for autonomous driving algorithms. To participate in the in-person competition, you must bring your own F1TENTH racecar. Detailed build instructions, including videos and step-by-step guidance, are available at [F1TENTH Build Instructions](https://f1tenth.org/build.html).

## Simulation

Heavy development in simulation is crucial for evaluating autonomous driving algorithms before deploying them on a physical vehicle. We offer various simulation environments:

*   **F1TENTH Gym:** This is the simulator used for the virtual competition. It's an asynchronous, 2D simulator built in Python that runs faster than real-time (30x). It features realistic vehicle simulation, collision detection, multi-vehicle support, and publishes laser scan and odometry data. [https://github.com/f1tenth/f1tenth_gym](https://github.com/f1tenth/f1tenth_gym)
*   **F1TENTH ROS Simulator:** For more in-depth vehicle development, this simulator provides ROS messages within a simulation environment. [https://github.com/f1tenth/f1tenth_simulator](https://github.com/f1tenth/f1tenth_simulator)

## Autonomous Racing

For those new to autonomous racing, we offer learning resources:

*   **F1TENTH Penn Course Material:** The complete material from the F1TENTH Penn course is available online at [F1TENTH Learn](https://f1tenth.org/learn.html). This course covers autonomous driving foundations, includes tutorials on the F1TENTH car, and explores autonomous racing techniques like raceline finding.
*   **F1TENTH Autonomous Racing Course Lectures:** All lectures from the course have been recorded and can be found on YouTube: [https://youtu.be/zENhppcxwzY](https://youtu.be/zENhppcxwzY)
