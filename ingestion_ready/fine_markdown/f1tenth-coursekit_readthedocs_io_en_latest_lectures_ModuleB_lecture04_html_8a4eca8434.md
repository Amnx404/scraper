# Lecture 4 - Laplace Domain Dynamics & PID

**Overview:**

We command the PID control on the VESC, which converts an RPM requirement to a current output for the motor. The PID controller ensures the correct current is supplied to achieve the desired RPM.

1.  VESC tuning
2.  Wall following: The error is the distance to the wall (left or right). We control the steering angle based on this error.
