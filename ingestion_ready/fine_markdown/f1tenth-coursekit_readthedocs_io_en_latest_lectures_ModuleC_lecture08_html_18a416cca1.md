# Lecture 8 - Localization: Particle Filter

**Overview:**

This lecture covers particle filters in general, and specifically their use in Monte Carlo localization for robots within a known environment.

**Topics Covered:**

*   Map making with Hector SLAM
*   Particle Filter Localization
*   Adaptive Monte Carlo Localization (AMCL)
*   Recursive Bayes Filtering used in AMCL
*   Tuning Particle Filters in ROS
