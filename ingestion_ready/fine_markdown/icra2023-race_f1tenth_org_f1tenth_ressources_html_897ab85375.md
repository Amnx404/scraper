# F1TENTH Resources

## Build

The F1TENTH Autonomous Vehicle System is an open-source platform for autonomous systems research and education on a 1:10 scale. It serves as the baseline for the in-person competition, providing sensors and computation power for autonomous driving algorithms. To participate in the in-person competition, you must bring your own F1TENTH racecar. Detailed build instructions, including videos and step-by-step guidance, are available at [F1TENTH Build Instructions](https://f1tenth.org/build.html).

## Simulation

Heavy development in simulation is crucial for evaluating autonomous driving algorithms before deploying them on a physical vehicle. We offer various simulation environments:

*   **F1TENTH Gym:** This asynchronous, 2D simulator built in Python is used for the virtual competition. It runs faster than real-time (30x), offers realistic vehicle simulation and collision detection, supports multiple vehicle instances, and publishes laser scan and odometry data. [F1TENTH Gym](https://github.com/f1tenth/f1tenth_gym)
*   **F1TENTH ROS Simulator:** This simulator provides ROS messages from the F1TENTH car within a simulation environment, useful for more in-depth vehicle development. [F1TENTH ROS Simulator](https://github.com/f1tenth/f1tenth_simulator)

## Autonomous Racing

For those new to autonomous racing, we provide learning resources:

*   **F1TENTH Penn Course Material:** The complete material from the F1TENTH Penn course, covering autonomous driving foundations, F1TENTH car tutorials, and autonomous racing techniques like raceline finding, is available online at [F1TENTH Learn](https://f1tenth.org/learn.html).
*   **F1TENTH Autonomous Racing Course (YouTube):** All lectures from the course have been recorded and can be found on YouTube: [F1TENTH Autonomous Racing Course](https://youtu.be/zENhppcxwzY).
