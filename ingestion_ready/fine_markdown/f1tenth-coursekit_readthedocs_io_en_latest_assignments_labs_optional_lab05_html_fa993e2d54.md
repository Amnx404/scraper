# Lab 5 - Scan Matching (Optional)

**Note:** This is a **GROUP** assignment.

**Tip:** Before starting this lab, review [Lecture 9](../../lectures/ModuleC/lecture09.html#doc-lecture09) and [Lecture 10](../../lectures/ModuleD/lecture10.html#doc-lecture10) to ensure you are familiar with the material.

**Goals:**

This lab addresses the problem of robot localization and introduces its importance in the autonomy stack. Through this lab, the fundamental scan matching algorithm, specifically the Iterative Closest Point (ICP) algorithm, will be implemented. You can refer to the [Andre Censi PLICP paper](https://censi.science/pub/research/2008-icra-plicp.pdf) for reference. By the end of this lab, you will gain knowledge and expertise in robot localization within a mapped environment and understand its significance in path planning and trajectory tracking.

**Learning Outcomes:**

Upon completion of this lab, students should understand the following fundamental concepts:

*   Localization
*   Odometry Estimation
*   Convex optimization
*   C++ OOP
*   Quadratic Programming

**Required Skills:** ROS, Python/C++
